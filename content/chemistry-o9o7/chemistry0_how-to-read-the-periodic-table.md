# Chemistry 0: How to Read the Periodic Table

## 1. The Big Idea (30 seconds)

- The periodic table looks confusing until you know what the rows and columns mean
- Once you know how shells fill up, you can work out what an atom will do
- That means you can predict bonding instead of just memorising it
- This is one of the most useful shortcuts in chemistry

---

## 2. What Rows and Columns Mean

Firstly:

- Periods (rows in the table) just mean how many shells (rings on the diagram) there are. It's worth remembering this.

![Periods show number of shells](../images/chemistry0_periods-mean-shells.svg)

- Groups (columns) just means how many electrons are in the outer shell

![Groups show outer electrons](../images/chemistry0_groups-mean-outer-electrons.svg)

Read that twice, or until you get it because that is a big clue. Then...

---

## 3. The Main Rule

- If an atom has one shell, it’s full at 2.
- If it has more shells, the outer one is full at 8.
- Say it with me. First shell 2, every other shell 8. Not very catchy but whatevs

So you have to fill the shells like this:

- 2 then next one is 8 then the one after is 8 until you get to one that isn't full.

---

## 4. What Happens Next

- If the outer shell isn't 8 (or 2 in the case of only one shell) it will react!
- Then if it will react, just take the easiest route to a complete shell.

So...

- If the outer shell is 1,2 or 3 you need to remove electrons.
- If it's 5,6 or 7 then you need to add electrons.
- (If it's 4 then it usually shares, so just park that for now)

---

## 5. Worked Examples

### Atomic number 3

![Atomic number 3 example](../images/chemistry0_atomic-number-3-example.svg)

So firstly you need to fill the inner shell, so 2 go there. Then only 1 remaining so that goes on the outer. 

- Fill the shells: 2,1
- Outer shell is not full
- Easiest route to stability => lose 1 electron

So:

- 2,1
- Reacts
- Loses 1 electron

### Atomic number 18

![Atomic number 18 example](../images/chemistry0_atomic-number-18-example.svg)

- Fill the shells: 2,8,8
- Outer shell is full

So:

- 2,8,8
- Already full
- Does nothing

### Atomic number 15

![Atomic number 15 example](../images/chemistry0_atomic-number-15-example.svg)

So firstly you need to fill the inner shell, so 2 go there. Then the next shell so 8 go there. Then 5 on the outer. 

- Fill the shells: 2,8,5
- Outer shell is not full
- Easiest route = gain 3

So:

- 2,8,5
- Reacts
- Gains 3 electrons

---

## 6. The Key Idea

- Fill 2, then 8s
- If the outer shell is full, it does nothing
- If it isn't full, do the easiest thing to make it full

That is basically the whole game.

---

## 7. Check Your Understanding

- What do periods (rows) tell you? ( how many shells there are )
- What do groups (columns) tell you? ( how many electrons are in the outer shell )
- If an atom has one shell, when is it full? ( 2 )
- If an atom has more than one shell, when is the outer shell full? ( 8 )
- What happens if the outer shell is not full? ( it reacts )
- If the outer shell has 2 electrons, what will usually happen? ( remove 2 )
- If the outer shell has 7 electrons, what will usually happen? ( add 1 )

---

## 8. Why This Matters

You will read about bonding in a minute.

What we now know is:

- shells contain electrons
- atoms react when the outer shell is not full
- bonding is just atoms trying to sort that out

That makes the next chapter a lot easier.

## What the numbers mean (quick finish)

Each box in the periodic table gives you a couple of key numbers.

- Atomic number = number of protons  
- This defines the element  
- It also tells you how many electrons there are (in a neutral atom)

So:

- Atomic number 3 → 3 protons → 3 electrons  
- That’s how we got 2,1 earlier  

Mass number:

- Protons + neutrons  
- You don’t need this much yet, but it explains why atoms have weight  

Key idea:

- Protons = what the element is  
- Electrons = how it behaves
